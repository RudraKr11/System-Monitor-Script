# 🖥️ System Monitor Script (macOS / Unix)

A Bash-based system monitoring tool that displays real-time system health including CPU, memory, disk usage, running processes, and active users.

---

## 🚀 Features

- 📊 CPU Usage Monitoring  
- 🧠 Memory Usage Calculation  
- 💾 Disk Usage Tracking  
- 🔥 Top Processes by CPU  
- 👥 Logged-in Users  
- ⚠️ Disk Usage Alerts  

---

## 🛠️ Tech Stack

- Bash Scripting
- Unix Commands (`top`, `vm_stat`, `df`, `ps`, `awk`, `sed`)

---

## ▶️ How to Run

```bash
chmod +x system_monitor.sh
./system_monitor.sh
